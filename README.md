# Soluciones del Codember 2023 de Midu

> Hecho con TypeScript y Jest

Para ejecutar el proyecto

```
//Instalar dependencias
npm i

//Ejecutar el codigo para solucionar el challenge #
npm start

//Ejecutar pruebas del challenge #
npm run test
```
